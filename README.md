# Bayan: AI Platform for Learning Saudi Sign Language

## Project Overview
Bayan is an AI-powered interactive learning platform designed to teach Saudi Sign Language (SSL). The platform bridges the communication gap between the deaf and mute community and the wider society by providing an accessible, engaging, and practical learning experience.

## Features
- AI-Powered Gesture Recognition 
  - Users can practice sign language gestures in real-time, with AI providing instant feedback.
- Interactive Learning Modules* 
  - Structured courses with quizzes, lessons, and practice sessions.
- Gesture Dictionary 
  - A comprehensive Saudi Sign Language dictionary with video demonstrations.
- Real-Time Feedback
  - AI evaluates user performance and suggests improvements.
- User Progress Tracking 
  - Users can monitor their learning journey and improvements over time.

## Problem Statement
Saudi Arabia has a significant communication gap due to the shortage of certified sign language interpreters. Current educational tools for sign language lack interactivity and practical application, making it difficult for learners to develop fluency.

Bayan provides a practical and engaging AI-powered learning experience, helping individuals learn SSL efficiently and effectively.

### AI & Machine Learning
- PyTorch – Deep learning framework for gesture recognition.
- Hugging Face Transformers – Pretrained models for video classification.
- OpenCV & Mediapipe – Video processing for real-time sign detection.

## Dataset
The dataset consists of 8,500 video samples of Saudi Sign Language gestures. Each video is preprocessed and augmented to improve model performance.

- Data Augmentation Techniques Used:
  - Frame extraction & resampling
  - Rotation & brightness adjustments

## Future Enhancements
- Expand the dataset to cover a broader range of SSL gestures.
- Improve AI model accuracy for better gesture recognition.
- Add a peer-to-peer interaction feature for practicing with real users.
